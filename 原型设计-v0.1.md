## 三、关键交互流程

### 流程1: 发起挑战到开始

```
首页 → 点击"发起挑战" → 填写参数 → 点击"邀请好友" 
→ 微信分享卡片 → 好友点击加入 → 双方确认 → 挑战开始
```

### 流程2: 每日打卡

```
首页 → 点击进行中的挑战 → 点击"去打卡" 
→ 填写体重 → (可选)记录饮食/运动 → 提交 → 查看PK结果
```

### 流程3: 挑战结束

```
达到挑战天数 → 自动结算 → 显示结果页 
→ 确认惩罚 → 分享海报 → 选择再来一局/结束
```

## 四、设计规范建议

### 配色方案

**情侣模式（粉色系）**
- 主色: #FF6B9D (粉红)
- 辅色: #FFE4EC (浅粉)
- 强调: #FF3366 (深红)

**朋友模式（蓝色系）**
- 主色: #4A90E2 (蓝)
- 辅色: #E8F4FD (浅蓝)
- 强调: #2E5C8A (深蓝)

**通用**
- 背景: #F5F5F5 (浅灰)
- 文字: #333333 (深灰) / #999999 (浅灰)
- 成功: #52C41A (绿)
- 警告: #FAAD14 (橙)

### 字体建议
- 标题: 18-20px 粗体
- 正文: 14-16px 常规
- 辅助: 12px 灰色
